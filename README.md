# Blinkit_Dashboard
Power BI dashboard built to analyze and visualize key business insights for Blinkit
Excited to share my latest Power BI dashboard built to analyze and visualize key business insights for Blinkit, one of India’s leading instant delivery platforms.
This dashboard delivers a data-driven view of operational and sales performance, enabling faster decision-making and strategic planning.

Tools Used: Power BI | Power Query | DAX | Data Modeling
